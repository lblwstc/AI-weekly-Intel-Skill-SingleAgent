# Output Contract

Use this skill when the user wants a one-shot internal AI intelligence product.

Required project structure:

```text
<project-root>/
  data/
    ai_updates.json
  content/
    weekly_ai_brief.md
  webapp/
  automation/
  demo/
```

Minimum outputs:

- `data/ai_updates.json`
- `content/weekly_ai_brief.md`
- A working dashboard app under `webapp/`
- At least one demo video under `demo/`
- A root `README.md`

Dataset requirements:

- 5 to 10 relevant updates
- Exact publication date
- Source name
- Source URL
- Summary
- Key technical capability
- Example use case
- Why it matters
- Business impact for knowledge workers

Memo requirements:

- Executive summary
- Top innovations this week
- Implications for enterprise productivity
- Example workflows
- Risks or limitations
- Strategic outlook

Dashboard requirements:

- Responsive layout
- Clear navigation
- Clickable or expandable update cards
- Search or filtering
- Insight and implication sections
- Internal strategy-team visual language

Demo requirements:

- Show navigation across the app
- Show update exploration
- Show insights and use cases
- Export an MP4
- If narrated, keep subtitle assets separate from the original video
